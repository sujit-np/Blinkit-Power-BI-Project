The project focuses on analyzing an online grocery shopping store called 'Blinkit'
Connect Power BI Desktop to an Excel sheet containing grocery store data 
Explores the data to understand its structure and content
My goal was to build an interactive and dynamic dashboard using various visuals like cards, line charts, and bar charts
Added filters and slicers to enable users to explore the data dynamically
Formatted elements like colors and fonts.
Established relationships between visuals to create a dynamic and interactive dashboard.
